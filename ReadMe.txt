
📦 DRILLING DASHBOARD PACKAGE – HOW TO USE

1. ✅ Requirements:
   - Install Python from https://www.python.org/
   - In Command Prompt, run:
     pip install streamlit pandas matplotlib

2. 📁 Files:
   - your_cleaned_data_sample.csv → Sample data
   - drilling_dashboard.py → Dashboard script
   - ReadMe.txt → This guide

3. ▶️ How to Run:
   - Open Command Prompt
   - Navigate to this folder (e.g., cd Downloads)
   - Run: streamlit run drilling_dashboard.py
   - Dashboard will open in your browser!
